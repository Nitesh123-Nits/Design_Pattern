Structural Design Patterns

Structural patterns explain how to assemble objects and classes into larger structures, while keeping these structures flexible and efficient.

Adapter

Converts incompatible interfaces. Plug legacy/third-party code into your system.

Easy

Bridge

Separates abstraction from implementation — both can vary independently.

Hard

Composite

Tree structures where leaves and branches are treated uniformly.

Medium

Decorator

Dynamically add responsibilities to objects. Alternative to subclassing.

Medium

Facade

Simplified API for complex subsystems. Used in every Spring service layer.

Easy

Flyweight

Share intrinsic state among many fine-grained objects to save memory.

Hard

Proxy

Surrogate object for access control, caching, lazy loading. How Spring AOP works.

Medium
